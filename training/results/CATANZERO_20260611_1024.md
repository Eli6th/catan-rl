# CatanZero 1024-wide rerun

## Configuration

- 10 training minutes on laptop CPU.
- 1024-wide model with 2.78 million parameters.
- Four parallel frozen self-play actors.
- Same 4,454 AlphaBot bootstrap decisions as the 192-wide run.
- Official hidden information and root re-determinization.
- Progressive 4/8/16-simulation MCTS curriculum.
- PR #1 sparse-column Rust inference.

Artifacts are under `training/runs/20260611-1024-catanzero-pr1/`.
The selected checkpoint is `checkpoints/game_0556.pt`.

## Verdict

Increasing width from 192 to 1024 produced a clear recovery against every
fixed baseline and beat the compact model directly. It did not recover
the strength of the previous 512-wide CatanZero checkpoint, losing all
24 direct holdout games.

Model capacity was a real constraint, but it was not the primary failure.
The remaining gap is consistent with insufficient teacher coverage and
weak greedy-policy calibration after low-budget search distillation. This
checkpoint should not replace the prior 512-wide champion.
